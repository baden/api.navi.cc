Системы (трекеры)

  • Поля ресурса system
  • Получение списка всех систем
  • Получение системы

Доступ к ресурсу /systems разрешен только если указанная система находится в списке наблюдения авторизованного пользователя. Также ресурс доступен для пользователей, состоящих в группе admin.

Запись системы создается автоматически при первом выходе на связь трекера.

Поля ресурса system

id
string - Идентификатор системы.
imei
string - IMEI GPRS-модуля трекера. По данному полю происходит добавление трекера в список наблюдения.
phone
string - Номер SIM-карты GPRS-модуля трекера. Если SIM-карта оператора содержит в телефонной книге запись собственного номера, то данное поле будет установлено автоматически.
date
dt - Дата регистрации (первого выхода на связь).
premium
dt - Дата окончания премиум-подписки трекера.
title
string - Отображаемое наименование транспортного средства.
icon
string - Значек транспортного средства.
car
object - Информация о транспортном средстве. Может содержать дополнительные записи о транспортном средстве.

Пример:

number
string - Гос номер
engine
string - № двигателя
vin
string - № кузова
insurance
string - Страховой полис
drivers
array of string - Список идентификаторов записей о водителях.
tags
array of string - Список ярлыков транспортного средства. По ярлыкам возможна сортировка списка трекеров.
groups
array of string - Список групп пользователей к которым прикреплено транспортное средство. Необходимо чтобы после добавления трекера в список наблюдения пользователем с полномочиями администратора группы, но не состоящим в группе admin, автоматически предлагалось добавление трекера в группу пользователя.
lock
boolean - Если это поле установлено в значение true, то трекер нельзя будет добавить в список наблюдения. Только администратор одной из групп из списка groups может менять значение этого поля.
public
boolean - Если это поле установлено в значение true, то трекер автоматически добавляется всем членам групп из списка groups если при добавлении трекера была нажата кнопка “Трекеры группы”.
hwid
string - Строка модели трекера.
swid
string - Версия программного обеспечения трекера.
params
object - Параметры трекера. Хранит параметры для различных блоков работы с трекером. Сервер не накладывает никаких специальных правил для хранения параметров.

Пример некоторых параметров:

fuel
array of object - Список записей калибровки датчика топлива

Каждая запись имеет формат:

voltage
float - Напряжение в вольтах
liters
float - Уровень топлива в баке
dynamic
object - Только для чтения. Динамически изменяемые данные. К каждому трекеры прикреплен специальный блок часто изменяемых данных. В этом блоке хранятся данные, которые изменяются при каждом обращении трекера к серверу. Для сохранения данного блока сервер использует оперативную память или базу с негарантированным сохранением, поэтому нужно быть готовым к тому, что данный блок не будет сохранен.

Перечень параметров (далее под “текущее” подразумевается значение при последнем выходе на связь):

dt
dt - Дата/время последнего выхода на связь
latitude
float - Широта текущего положения
longitude
float - Долгота текущего положения
speed
float - Текущая скорость
course
float - Текущее направление движения
altitude
integer - Текущая высота над уровнем моря
sats
integer - Число спутников
vout
float - Напряжение основного питания
vin
float - Напряжение резервного питания
fsource

integer - Тип последней точки.

fuel
float - Значение АЦП-входа, к которому подключен датчик топлива
temp
float - Значение АЦП-входа, к которому подключен датчик температуры
adc1
float - Значение АЦП-входа 1
adc2
float - Значение АЦП-входа 2
csq
integer - Уровень сигнала GSM-сети, определяемый формулой (csq * 2 - 113) дБ.
gps
float - Усредненный уровень сигнала спутников.
in1
boolean - Состояние логического входа 1
in2
boolean - Состояние логического входа 2
in3
boolean - Состояние логического входа 3
out1
boolean - Состояние выхода 1
out2
boolean - Состояние выхода 2

Получение списка всех систем

GET /systems

Ответ

Status: 200 OK
[
  "simple_system1",
  "simple_system2",
  "simple_system3"
]
